# VScode
- Generate markdown table of content
  ```
  ctrl-shift-p
  select "Create table of contents"
  ```

- add 2 space at the end of the list item and then add a new line to move the cursor to next line to the list item  
  for example
  ```markdown
  - abc[space][space]
    ads
  ```  
  will look like this
  - abc  
    ads
